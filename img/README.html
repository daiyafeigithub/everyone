图片资源目录

此目录用于存放博客的图片资源。

必需文件

1. favicon.png

用途: 网站图标（浏览器标签页显示）
尺寸: 16x16px 或 32x32px 或 64x64px
格式: PNG（支持透明背景）

快速生成:

  • 在线工具: https://realfavicongenerator.net/
  • 或使用在线转换: https://favicon.io/

2. avatar.png

用途: 个人头像
尺寸: 200x200px 或更大（建议正方形）
格式: PNG 或 JPG

示例:

  • 使用自己的照片
  • 或使用卡通头像
  • 或从 https://www.flaticon.com/ 下载图标

可选文件

背景图片

  • default_bg.jpg - 默认背景图（1920x1080px）
  • index_bg.jpg - 首页背景
  • archive_bg.jpg - 归档页背景
  • tag_bg.jpg - 标签页背景
  • category_bg.jpg - 分类页背景

文章封面

  • cover1.jpg ~ cover10.jpg - 文章封面图（1920x1080px）

临时解决方案

如果暂时没有图片，可以：

  1. 下载免费图片:

    • https://unsplash.com/
    • https://pixabay.com/
    • https://pexels.com/

  2. 使用在线生成器:

    • Favicon: https://favicon.io/favicon-generator/
    • 占位图: https://placehold.co/

  3. 使用Emoji作为favicon:

    • 访问: https://favicon.io/emoji-favicons/
    • 选择喜欢的emoji
    • 下载并重命名为 favicon.png

注意事项

  1. 图片文件名不要使用中文
  2. 建议使用压缩后的图片以提升加载速度
  3. 大图建议使用 JPG 格式，小图标使用 PNG
  4. 本目录的文件会被复制到 public/img/ 目录

  • 在线压缩: https://tinypng.com/
  • 批量压缩: https://squoosh.app/
