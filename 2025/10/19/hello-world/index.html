дёҖгҖҒжӮІи§Ӯй”Ғзҡ„з”ұжқҘпјҡд»Һ вҖңи¶…еҚ–дәӢж•…вҖқ иҜҙиө·

е’ұд»¬е…Ҳд»ҺдёӨдёӘеңәжҷҜеҲҮе…ҘпјҢзңӢжҮӮдёәе•ҘйңҖиҰҒжӮІи§Ӯй”Ғпјҡ

1. з”ҹжҙ»еңәжҷҜпјҡеҚ«з”ҹй—ҙзҡ„ вҖңе№¶еҸ‘еҶІзӘҒвҖқ

дҪ еҺ»е…¬еҸёеҚ«з”ҹй—ҙпјҢжҺЁй—ЁеҸ‘зҺ°жІЎдәәдҪҶжІЎй”Ғй—Ё вҖ”вҖ” иҝҷж—¶еҖҷеҰӮжһңзӘҒз„¶еҶІиҝӣжқҘеҸҰдёҖдёӘдәәпјҢе°ұдјҡеҸ‘з”ҹ вҖңиө„жәҗеҶІзӘҒвҖқгҖӮжүҖд»ҘиҒӘжҳҺзҡ„еҒҡжі•жҳҜ е…ҲеҸҚй”ҒеҶҚдҪҝз”Ёпјҡй»ҳи®Ө вҖңдёҖе®ҡдјҡжңүдәәжҠўвҖқпјҢжҸҗеүҚй”Ғжӯ»иө„жәҗпјҢиҮӘе·ұз”Ёе®ҢеҶҚйҮҠж”ҫгҖӮ

2. зј–зЁӢеңәжҷҜпјҡдёҚеҠ й”Ғзҡ„ вҖңеә“еӯҳзҒҫйҡҫвҖқ

еҒҮи®ҫз”өе•Ҷз§’жқҖжҹҗе•Ҷе“ҒпјҢеә“еӯҳ 100 д»¶пјҢ1000 дёӘз”ЁжҲ·еҗҢж—¶дёӢеҚ•гҖӮеҰӮжһңдёҚеҠ й”ҒпјҢд»Јз ҒеҸҜиғҪеҶҷжҲҗиҝҷж ·пјҡ

иҝҗиЎҢз»“жһңзҒҫйҡҫпјҡ1000 дёӘзәҝзЁӢеҗҢж—¶жү§иЎҢж—¶пјҢдјҡеҮәзҺ° вҖңеә“еӯҳеҸҳжҲҗиҙҹж•°вҖқ зҡ„и¶…еҚ– вҖ”вҖ” еӣ дёәеӨҡдёӘзәҝзЁӢеҗҢж—¶йҖҡиҝҮжӯҘйӘӨ 1 зҡ„еҲӨж–ӯпјҢйғҪи®Өдёәеә“еӯҳе……и¶іпјҢжңҖеҗҺеҗҢж—¶жү§иЎҢжӯҘйӘӨ 2 зҡ„жүЈеҮҸгҖӮ

иҝҷе°ұжҳҜ е№¶еҸ‘еҶІзӘҒпјҡеӨҡдёӘзәҝзЁӢеҗҢж—¶ж“ҚдҪңе…ұдә«иө„жәҗпјҲеә“еӯҳпјүпјҢеҜјиҮҙж•°жҚ®ж··д№ұгҖӮжӮІи§Ӯй”Ғзҡ„иҜһз”ҹе°ұжҳҜдёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҡж“ҚдҪңеүҚе…ҲдёҠй”ҒпјҢеҗҢдёҖж—¶й—ҙеҸӘе…Ғи®ёдёҖдёӘзәҝзЁӢеҠЁжүӢпјҢеғҸеҚ«з”ҹй—ҙеҸҚй”ҒдёҖж ·йҳ»ж–ӯеҶІзӘҒгҖӮ

дәҢгҖҒжӮІи§Ӯй”Ғзҡ„ж ёеҝғжҰӮеҝөпјҡ3 дёӘзү№зӮ№и®ІйҖҸ

жӮІи§Ӯй”ҒжҳҜдёҖз§Қ вҖңе…ҲйҳІеҫЎеҗҺж“ҚдҪңвҖқ зҡ„е№¶еҸ‘зӯ–з•ҘпјҢе®ғй»ҳи®Ө вҖңзәҝзЁӢдёҖе®ҡдјҡжҠўиө„жәҗвҖқпјҢжүҖд»ҘеңЁи®ҝй—®е…ұдә«ж•°жҚ®еүҚдё»еҠЁиҺ·еҸ–й”ҒпјҢзЎ®дҝқеҸӘжңүжҢҒй”ҒзәҝзЁӢиғҪж“ҚдҪңпјҢе…¶д»–зәҝзЁӢеҝ…йЎ»жҺ’йҳҹзӯүеҫ…гҖӮ

е…ій”®зү№зӮ№з”Ё вҖңеҚ«з”ҹй—ҙжЁЎеһӢвҖқ зұ»жҜ”пјҡ

зү№зӮ№ жҠҖжңҜи§ЈйҮҠ з”ҹжҙ»еҜ№еә”еңәжҷҜ
зӢ¬еҚ жҖ§ й”Ғиў«дёҖдёӘзәҝзЁӢжҢҒжңүеҗҺпјҢе…¶д»–зәҝзЁӢж— жқғдҪҝз”Ё еҚ«з”ҹй—ҙеҸҚй”ҒеҗҺпјҢеҸӘжңүйҮҢйқўзҡ„дәәиғҪдҪҝз”Ё
йҳ»еЎһжҖ§ жІЎжӢҝеҲ°й”Ғзҡ„зәҝзЁӢжҡӮеҒңжү§иЎҢпјҢдёҚдјҡеҚ з”Ё CPU жІЎжҠўеҲ°еҚ«з”ҹй—ҙзҡ„дәәеңЁй—ЁеҸЈжҺ’йҳҹзӯүеҫ…
ејәдёҖиҮҙжҖ§ иғҪ 100% йҒҝе…ҚеҶІзӘҒпјҢдҪҶдјҡзүәзүІжҖ§иғҪ иҷҪз„¶жҺ’йҳҹиҙ№ж—¶й—ҙпјҢдҪҶз»қеҜ№дёҚдјҡж’һиҪҰ

дёүгҖҒJava дёӯжӮІи§Ӯй”Ғзҡ„е®һзҺ°ж–№ејҸпјҡ2 з§Қй”Ғе·Ҙе…·ж·ұеәҰи§Јжһҗ

Java йҮҢжӮІи§Ӯй”ҒеҲҶ вҖңйҡҗејҸй”ҒвҖқпјҲsynchronizedпјүе’Ң вҖңжҳҫејҸй”ҒвҖқпјҲReentrantLockпјүпјҢдёӨиҖ…еҺҹзҗҶе’Ңз”Ёжі•е·®ејӮеҫҲеӨ§пјҢйҮҚзӮ№з”ЁеӣҫзӨәжӢҶи§ЈйҡҫзӮ№гҖӮ

1. йҡҗејҸй”ҒпјҡsynchronizedпјҲжҮ’дәәдё“еұһпјҢJVM её®дҪ з®Ўй”Ғпјү

synchronized жҳҜ Java иҮӘеёҰзҡ„ вҖңеӮ»з“ңејҸй”ҒвҖқпјҢдёҚз”ЁжүӢеҠЁејҖе…ій”ҒпјҢJVM иҮӘеҠЁеӨ„зҗҶдёҖеҲҮгҖӮдҪҶеә•еұӮзҡ„ MonitorпјҲзӣ‘и§ҶеҷЁй”ҒпјүжңәеҲ¶ жҳҜйҡҫзӮ№пјҢеҝ…йЎ»зңӢеӣҫзҗҶи§ЈгҖӮ

пјҲ1пјүMonitor жңәеҲ¶еҸҜи§ҶеҢ–и§ЈжһҗпјҲж ёеҝғйҡҫзӮ№пјҒпјү

Monitor жң¬иҙЁжҳҜдёӘ вҖңз®ЎзЁӢжЁЎеһӢвҖқпјҢеғҸдёӘеёҰй—ЁеҚ«зҡ„жҲҝй—ҙпјҢе…ұдә«иө„жәҗпјҲжҜ”еҰӮеә“еӯҳеҸҳйҮҸпјүж”ҫеңЁжҲҝй—ҙйҮҢпјҢеҸӘжңүжӢҝеҲ°й’ҘеҢҷзҡ„зәҝзЁӢиғҪиҝӣгҖӮ

е…ій”®йҖ»иҫ‘пјҲеҜ№еә” synchronized жү§иЎҢжөҒзЁӢпјүпјҡ

  1. зәҝзЁӢиҝӣе…Ҙ synchronized д»Јз Ғеқ—ж—¶пјҢдјҡеҗ‘ Monitor з”іиҜ· вҖңй’ҘеҢҷвҖқпјҲеҠ й”Ғпјүпјӣ

  2. еҰӮжһң Monitor жІЎдәәжҢҒжңүпјҲowner дёәз©әпјүпјҢзәҝзЁӢзӣҙжҺҘжҲҗдёә ownerпјҢзӢ¬еҚ иө„жәҗпјӣ

  3. еҰӮжһңе·ІжңүзәҝзЁӢжҢҒжңүй’ҘеҢҷпјҲжҜ”еҰӮзәҝзЁӢ AпјүпјҢж–°зәҝзЁӢпјҲзәҝзЁӢ Cпјүдјҡиҝӣе…Ҙ entrySet йҳҹеҲ— йҳ»еЎһзӯүеҫ…пјӣ

  4. зәҝзЁӢжү§иЎҢе®Ңд»Јз Ғеқ—пјҢжҲ–еҸ‘з”ҹејӮеёёпјҢJVM дјҡиҮӘеҠЁйҮҠж”ҫй’ҘеҢҷпјҲowner зҪ®з©әпјүпјҢе№¶е”ӨйҶ’ entrySet йҮҢзҡ„дёҖдёӘзәҝзЁӢжҠўй”Ғпјӣ

  5. еҰӮжһңзәҝзЁӢеңЁй”ҒеҶ…и°ғз”Ё wait() ж–№жі•пјҲжҜ”еҰӮзӯүеҫ…е…¶д»–зәҝзЁӢйҖҡзҹҘпјүпјҢдјҡжҡӮж—¶йҮҠж”ҫй’ҘеҢҷпјҢиҝӣе…Ҙ waitSet йҳҹеҲ—пјҢиў« notify() е”ӨйҶ’еҗҺеҶҚеӣһ entrySet жҺ’йҳҹгҖӮ

пјҲ2пјүsynchronized зҡ„ 3 з§Қз”Ёжі•пјҲд»Јз Ғе®һжҲҳпјү

2. жҳҫејҸй”ҒпјҡReentrantLockпјҲзҒөжҙ»зҺӢиҖ…пјҢжүӢеҠЁжҺ§й”Ғпјү

Java 1.5 ж–°еўһзҡ„ java.util.concurrent.locks.Lock жҺҘеҸЈе®һзҺ°зұ»пјҢйңҖиҰҒжүӢеҠЁ lock() еҠ й”ҒгҖҒunlock() йҮҠж”ҫй”ҒпјҢеҠҹиғҪжҜ” synchronized ејә 10 еҖҚгҖӮж ёеҝғйҡҫзӮ№жҳҜ AQS жЎҶжһ¶еҺҹзҗҶгҖӮ

пјҲ1пјүAQS жЎҶжһ¶еҸҜи§ҶеҢ–и§ЈжһҗпјҲеҝ…жҮӮпјҒпјү

ReentrantLock еҹәдәҺ AQSпјҲAbstractQueuedSynchronizerпјү е®һзҺ°пјҢжң¬иҙЁжҳҜдёӘ вҖңеёҰдјҳе…Ҳзә§зҡ„жҺ’йҳҹзі»з»ҹвҖқпјҢз”ЁеҸҢеҗ‘й“ҫиЎЁеӯҳеӮЁзӯүеҫ…зәҝзЁӢпјҢйҖҡиҝҮ CAS ж“ҚдҪңпјҲCompare And SwapпјүжҺ§еҲ¶й”Ғзҡ„иҺ·еҸ–йҮҠж”ҫгҖӮ

е…ій”®йҖ»иҫ‘пјҡ

  1. зәҝзЁӢи°ғз”Ё lock() ж—¶пјҢAQS е…ҲжЈҖжҹҘ state еҸҳйҮҸпјҲ0 = ж— й”ҒпјҢ>0 = жңүй”Ғпјүпјӣ

  2. еҰӮжһң state=0пјҢз”Ё CAS ж“ҚдҪңжҠҠ state ж”№жҲҗ 1пјҢзәҝзЁӢжҲҗдёәжҢҒй”ҒзәҝзЁӢпјӣ

  3. еҰӮжһң stateвү 0пјҲй”Ғиў«еҚ з”ЁпјүпјҢзәҝзЁӢе°ҒиЈ…жҲҗ Node иҠӮзӮ№ еҠ е…Ҙзӯүеҫ…йҳҹеҲ—пјҢиҝӣе…Ҙйҳ»еЎһзҠ¶жҖҒпјӣ

  4. жҢҒй”ҒзәҝзЁӢи°ғз”Ё unlock() ж—¶пјҢжҠҠ state ж”№еӣһ 0пјҢе”ӨйҶ’йҳҹеҲ—еӨҙиҠӮзӮ№зҡ„зәҝзЁӢйҮҚж–°жҠўй”ҒгҖӮ

пјҲ2пјүReentrantLock зҡ„ж ёеҝғдјҳеҠҝпјҲд»Јз Ғе®һжҲҳпјү

жҜ” synchronized еӨҡ 3 дёӘе…ій”®еҠҹиғҪпјҢе…ЁжҳҜи§ЈеҶіе®һйҷ…й—®йўҳзҡ„пјҡ

пјҲ3пјүдёӨз§Қй”Ғзҡ„ж ёеҝғеҢәеҲ«пјҲеҸҜи§ҶеҢ–еҜ№жҜ”иЎЁпјү
з»ҙеәҰ synchronized ReentrantLock
й”Ғз®ЎзҗҶ йҡҗејҸпјҲJVM иҮӘеҠЁеҠ и§Јй”Ғпјү жҳҫејҸпјҲжүӢеҠЁ lock/unlockпјү
еҠҹиғҪ д»…еҹәзЎҖеҗҢжӯҘ ж”ҜжҢҒи¶…ж—¶гҖҒдёӯж–ӯгҖҒе…¬е№ій”Ғ
еҸҜйҮҚе…ҘжҖ§ ж”ҜжҢҒпјҲиҮӘеҠЁи®Ўж•°пјү ж”ҜжҢҒпјҲжүӢеҠЁи®Ўж•°пјҢйңҖеҢ№й…ҚйҮҠж”ҫж¬Ўж•°пјү
жҖ§иғҪ JDK1.6 еҗҺдјҳеҢ–пјҢдёҺ Lock жҺҘиҝ‘ й«ҳе№¶еҸ‘дёӢз•ҘдјҳпјҢеҠҹиғҪжӣҙзҒөжҙ»
йҖӮз”ЁеңәжҷҜ з®ҖеҚ•еҗҢжӯҘпјҲеҚ•ж–№жі• / д»Јз Ғеқ—пјү еӨҚжқӮеңәжҷҜпјҲз§’жқҖгҖҒеҲҶеёғејҸдәӢеҠЎпјү

еӣӣгҖҒжӮІи§Ӯй”Ғзҡ„еә”з”ЁеңәжҷҜпјҡ3 зұ»еңәжҷҜеҝ…йЎ»з”Ё

дёҚжҳҜжүҖжңүе№¶еҸ‘йғҪйңҖиҰҒжӮІи§Ӯй”ҒпјҢиҝҷ 3 з§Қжғ…еҶөдјҳе…ҲйҖүпјҡ

1. й«ҳеҶІзӘҒеңәжҷҜпјҡз§’жқҖгҖҒжҠўиҙӯ

зү№зӮ№пјҡ1000 дёӘзәҝзЁӢжҠў 100 дёӘиө„жәҗпјҢеҶІзӘҒжҰӮзҺҮжһҒй«ҳгҖӮ

жЎҲдҫӢпјҡз”өе•Ҷз§’жқҖеә“еӯҳжүЈеҮҸпјҢз”Ё ReentrantLock еҠ й”ҒпјҢй…ҚеҗҲи¶…ж—¶жңәеҲ¶йҒҝе…ҚзәҝзЁӢжӢҘе өгҖӮ

д»Јз ҒзүҮж®өпјҡ

2. ејәдёҖиҮҙжҖ§еңәжҷҜпјҡйҮ‘иһҚдәӨжҳ“

зү№зӮ№пјҡж•°жҚ®й”ҷ 1 еҲҶй’ұйғҪдёҚиЎҢпјҢеҝ…йЎ» 100% еҮҶзЎ®гҖӮ

жЎҲдҫӢпјҡй“¶иЎҢиҪ¬иҙҰпјҲA иҪ¬ B 100 е…ғпјүпјҢйңҖдҝқиҜҒ вҖңA жүЈй’ұвҖқ е’Ң вҖңB еҠ й’ұвҖқ еҺҹеӯҗжү§иЎҢпјҢз”Ё synchronized жҲ–ж•°жҚ®еә“жӮІи§Ӯй”ҒгҖӮ

ж•°жҚ®еә“жӮІи§Ӯй”ҒзӨәдҫӢпјҲй…ҚеҗҲ Java дәӢеҠЎпјүпјҡ

3. й•ҝж“ҚдҪңеңәжҷҜпјҡжү№йҮҸж•°жҚ®еӨ„зҗҶ

зү№зӮ№пјҡж“ҚдҪңиҖ—ж—¶д№…пјҲжҜ”еҰӮжү№йҮҸеҜје…Ҙ 1000 жқЎж•°жҚ®пјүпјҢжңҹй—ҙе®№жҳ“иў«е…¶д»–зәҝзЁӢжү“ж–ӯгҖӮ

жЎҲдҫӢпјҡExcel жү№йҮҸеҜје…Ҙз”ЁжҲ·ж•°жҚ®пјҢйңҖдҝқиҜҒеҜје…ҘиҝҮзЁӢдёӯж•°жҚ®дёҚиў«зҜЎж”№пјҢз”Ё ReentrantLock еҠ й”ҒпјҢйҒҝе…ҚдёӯйҖ”еҶІзӘҒгҖӮ

дә”гҖҒдҪҝз”ЁжӮІи§Ӯй”Ғзҡ„жіЁж„ҸдәӢйЎ№пјҡ4 дёӘеқ‘еҲ«иё©

1. еқ‘ 1пјҡй”ҒиҢғеӣҙиҝҮеӨ§пјҲвҖңй”ҒиҶЁиғҖвҖқпјү

й”ҷиҜҜзӨәиҢғпјҡжҠҠж•ҙдёӘж–№жі•еҠ й”ҒпјҢеҢ…жӢ¬ж— е…ізҡ„жҹҘиҜўж“ҚдҪңпјҡ

жӯЈзЎ®еҒҡжі•пјҡеҸӘй”Ғж ёеҝғд»Јз Ғеқ—пјҲй”ҒзІ’еәҰжңҖе°ҸеҢ–пјүпјҡ

2. еқ‘ 2пјҡжӯ»й”ҒпјҲжңҖиҮҙе‘ҪпјҒпјү

жӯ»й”Ғзҡ„ 4 дёӘеҝ…иҰҒжқЎд»¶пјҲзјәдёҖдёҚеҸҜпјҢз ҙеқҸд»»дҪ•дёҖдёӘеҚіеҸҜйҒҝе…Қпјүпјҡ

жӯ»й”ҒеҪўжҲҗеӣҫзӨәпјҡ

йҒҝе…Қж–№жі•пјҡ

  • з»ҹдёҖй”ҒйЎәеәҸпјҡжүҖжңүзәҝзЁӢйғҪе…ҲжӢҝ LOCK_AпјҢеҶҚжӢҝ LOCK_BпјҲз ҙеқҸ вҖңеҫӘзҺҜзӯүеҫ…вҖқпјүпјӣ

  • и¶…ж—¶йҮҠж”ҫпјҡз”Ё ReentrantLock зҡ„ tryLock(timeout)пјҲз ҙеқҸ вҖңжҢҒжңүе№¶зӯүеҫ…вҖқпјүпјӣ

  • дё»еҠЁдёӯж–ӯпјҡз”Ё lockInterruptibly() е“Қеә”дёӯж–ӯпјҲз ҙеқҸ вҖңдёҚеҸҜеүҘеӨәвҖқпјүгҖӮ

жӯ»й”ҒжЈҖжөӢе·Ҙе…·пјҡJDK иҮӘеёҰзҡ„ jconsoleпјҢжӯҘйӘӨеҰӮдёӢпјҡ

  1. еҗҜеҠЁзЁӢеәҸпјҢжү“ејҖ JDK зҡ„ bin зӣ®еҪ•пјҢеҸҢеҮ» jconsole.exeпјӣ

  2. йҖүжӢ©иҰҒзӣ‘жҺ§зҡ„ Java иҝӣзЁӢпјҢзӮ№еҮ» вҖңиҝһжҺҘвҖқпјӣ

  3. еҲҮжҚўеҲ° вҖңзәҝзЁӢвҖқ ж ҮзӯҫпјҢзӮ№еҮ» вҖңжЈҖжөӢжӯ»й”ҒвҖқпјҢеҚіеҸҜзңӢеҲ°жӯ»й”ҒзәҝзЁӢдҝЎжҒҜгҖӮ

3. еқ‘ 3пјҡеҝҪз•ҘеҸҜйҮҚе…ҘжҖ§зҡ„и®Ўж•°й—®йўҳ

ReentrantLock жүӢеҠЁеҠ й”Ғ n ж¬ЎпјҢеҝ…йЎ»йҮҠж”ҫ n ж¬ЎпјҢеҗҰеҲҷй”ҒдёҚдјҡйҮҠж”ҫпјҡ

4. еқ‘ 4пјҡй«ҳе№¶еҸ‘дёӢзҡ„жҖ§иғҪз“¶йўҲ

жӮІи§Ӯй”ҒдјҡеҜјиҮҙзәҝзЁӢйҳ»еЎһгҖҒдёҠдёӢж–ҮеҲҮжҚўпјҲзәҝзЁӢд»ҺиҝҗиЎҢеҲ°йҳ»еЎһеҶҚе”ӨйҶ’зҡ„ејҖй”ҖпјүгҖӮеҰӮжһңе№¶еҸ‘йҮҸжһҒй«ҳдҪҶеҶІзӘҒе°‘пјҲжҜ”еҰӮжҹҘиҜўеӨҡгҖҒдҝ®ж”№е°‘зҡ„з”ЁжҲ·еҲ—иЎЁпјүпјҢз”ЁжӮІи§Ӯй”ҒдјҡжөӘиҙ№ CPUпјҢжӯӨж—¶еә”жҚў д№җи§Ӯй”ҒпјҲз”Ё version еӯ—ж®өеҲӨж–ӯж•°жҚ®жҳҜеҗҰиў«дҝ®ж”№пјүгҖӮ

е…ӯгҖҒе…ёеһӢжЎҲдҫӢпјҡд»Һ вҖңи¶…еҚ–вҖқ еҲ° вҖңжӯ»й”Ғдҝ®еӨҚвҖқ е…Ёе®һжҲҳ

жЎҲдҫӢ 1пјҡsynchronized и§ЈеҶіеә“еӯҳи¶…еҚ–пјҲж–°жүӢе…Ҙй—Ёпјү

йңҖжұӮпјҡ100 дёӘзәҝзЁӢеҗҢж—¶жүЈеҮҸеә“еӯҳпјҢйҒҝе…Қи¶…еҚ–гҖӮ

е®Ңж•ҙд»Јз Ғ + иҝҗиЎҢеҲҶжһҗпјҡ

иҝҗиЎҢз»“жһңпјҡзәҝзЁӢдҫқж¬Ўжү§иЎҢпјҢеә“еӯҳд»Һ 100 еҮҸеҲ° 0пјҢж— и¶…еҚ–гҖӮеӣ дёә synchronized дҝқиҜҒдәҶ deduct ж–№жі•еҗҢдёҖж—¶й—ҙеҸӘжңүдёҖдёӘзәҝзЁӢиғҪжү§иЎҢгҖӮ

жЎҲдҫӢ 2пјҡReentrantLock е®һзҺ°з§’жқҖи¶…ж—¶жҺ§еҲ¶пјҲиҝӣйҳ¶пјү

йңҖжұӮпјҡз§’жқҖж—¶пјҢзәҝзЁӢжҠўй”Ғи¶…иҝҮ 2 з§’е°ұж”ҫејғпјҢйҒҝе…ҚеӨ§йҮҸзәҝзЁӢйҳ»еЎһгҖӮ

ж ёеҝғд»Јз Ғпјҡ

иҝҗиЎҢз»“жһңпјҡйғЁеҲҶзәҝзЁӢжҠўй”ҒжҲҗеҠҹпјҢйғЁеҲҶи¶…ж—¶ж”ҫејғпјҢйҒҝе…ҚдәҶжүҖжңүзәҝзЁӢйҳ»еЎһзӯүеҫ…гҖӮ

жЎҲдҫӢ 3пјҡжӯ»й”Ғдҝ®еӨҚпјҲеҸҚйқўвҶ’жӯЈйқўпјү

еҸҚйқўжЎҲдҫӢпјҡдёӨдёӘзәҝзЁӢдә’зӣёжҠўй”ҒпјҢеҜјиҮҙжӯ»й”ҒпјҲеҺҹд»Јз Ғи§ҒеүҚж–ҮпјүгҖӮ

дҝ®еӨҚж–№жЎҲпјҡз»ҹдёҖй”Ғзҡ„иҺ·еҸ–йЎәеәҸпјҢйғҪе…ҲжӢҝ LOCK_AпјҢеҶҚжӢҝ LOCK_BгҖӮ

дҝ®еӨҚеҗҺд»Јз Ғпјҡ

иҝҗиЎҢз»“жһңпјҡзәҝзЁӢ 1 е…ҲжӢҝеҲ° AпјҢжү§иЎҢе®ҢйҮҠж”ҫеҗҺпјҢзәҝзЁӢ 2 жӢҝеҲ° AпјҢдёҚдјҡжӯ»й”ҒпјҲз ҙеқҸдәҶ вҖңеҫӘзҺҜзӯүеҫ…вҖқ жқЎд»¶пјүгҖӮ

дёғгҖҒжҖ»з»“пјҡжӮІи§Ӯй”Ғзҡ„ вҖңз”ЁжӯҰд№Ӣең°вҖқ дёҺ вҖңйҒҝеқ‘жҢҮеҚ—вҖқ

1. д»Җд№Ҳж—¶еҖҷз”Ёпјҹ

  • вң… еҶІзӘҒеӨҡпјҲз§’жқҖгҖҒжҠўиҙӯпјүпјӣ

  • вң… дёҖиҮҙжҖ§иҰҒжұӮй«ҳпјҲйҮ‘иһҚгҖҒж”Ҝд»ҳпјүпјӣ

  • вң… ж“ҚдҪңж—¶й—ҙй•ҝпјҲжү№йҮҸеӨ„зҗҶгҖҒй•ҝдәӢеҠЎпјүгҖӮ

2. йҖү synchronized иҝҳжҳҜ ReentrantLockпјҹ

  • з®ҖеҚ•еңәжҷҜпјҲеҚ•ж–№жі•еҗҢжӯҘпјүвҶ’ з”Ё synchronizedпјҲзңҒеҝғпјҢдёҚжҳ“й”ҷпјүпјӣ

  • еӨҚжқӮеңәжҷҜпјҲи¶…ж—¶гҖҒдёӯж–ӯгҖҒе…¬е№ій”ҒпјүвҶ’ з”Ё ReentrantLockпјҲзҒөжҙ»пјҢеҠҹиғҪејәпјүгҖӮ

3. ж ёеҝғйҒҝеқ‘еҸЈиҜҖпјҡ

вҖңй”ҒзІ’еәҰиҰҒе°ҸпјҢйЎәеәҸиҰҒз»ҹдёҖпјҢи¶…ж—¶еҠ дёӯж–ӯпјҢйҮҠж”ҫиҰҒзүўи®°вҖқгҖӮ
